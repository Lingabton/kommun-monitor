🏛️ KOMMUN MONITOR

API Documentation

v1.0.0 — Gratis tillgång till strukturerade kommunbeslut

Base URL: https://kommun-monitor.se/api/v1

Snabbstart

API:et är statiskt — inga API-nycklar, ingen autentisering, inga rate limits. Hämta JSON direkt:

# Alla beslut
curl https://kommun-monitor.se/api/v1/decisions.json

# Ett specifikt beslut
curl https://kommun-monitor.se/api/v1/decisions/kf389.json

# Partistatistik
curl https://kommun-monitor.se/api/v1/parties.json

# JavaScript
const res = await fetch("https://kommun-monitor.se/api/v1/decisions.json");
const data = await res.json();
console.log(`${data.count} beslut`);

Endpoints

GET/meta.jsonGRATIS

API-metadata: version, statistik, tillgängliga endpoints.

GET/municipalities.jsonGRATIS

Lista över kommuner med data tillgänglig.

GET/meetings.jsonGRATIS

Alla sammanträden (sammanfattning). Sorterade efter datum (nyast först).

GET/meetings/{id}.jsonGRATIS

Enskilt sammanträde med alla beslut. Exempel: /meetings/2025-12-10_kf.json

GET/decisions.jsonGRATIS

Alla beslut som flat lista (headline, summary, category, date, tags). Utan full detail.

GET/decisions/{id}.jsonGRATIS

Enskilt beslut med full detalj: voting, quotes, tags, protocol reference.

GET/parties.jsonGRATIS

Partistatistik: röstningsfördelning, allianser, nyckelfrågor.

GET/parties/{abbr}.jsonGRATIS

Enskilt parti. Exempel: /parties/v.json, /parties/sd.json

GET/search.jsonGRATIS

Fulltextsökindex för client-side search. Varje entry innehåller normaliserat text-fält.

GET/feed.jsonGRATIS

JSON Feed (RFC) — för RSS-läsare och appar som stöder JSON Feed.

Användningsexempel

Python

import requests

# Hämta alla omstridda beslut
data = requests.get("https://kommun-monitor.se/api/v1/decisions.json").json()
contested = [d for d in data["decisions"] if d["contested"]]
print(f"{len(contested)} omstridda beslut")
for d in contested:
    print(f"  {d['date']} — {d['headline']}")

JavaScript

// Hämta partistatistik
const res = await fetch("https://kommun-monitor.se/api/v1/parties.json");
const { parties } = await res.json();

// Mest oppositionella parti
const mostOpposed = parties.sort((a,b) => b.against_pct - a.against_pct)[0];
console.log(`${mostOpposed.name} röstar NEJ ${mostOpposed.against_pct}% av gångerna`);

curl + jq

# Alla beslut om skola
curl -s https://kommun-monitor.se/api/v1/decisions.json | \
  jq '.decisions[] | select(.category == "skola") | .headline'

# Hur röstade SD?
curl -s https://kommun-monitor.se/api/v1/parties/sd.json | \
  jq '{name, for_pct, against_pct, top_allies}'
